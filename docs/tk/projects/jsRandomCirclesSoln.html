Yeniden Tesadüfi Daireler

Hazırlayanlar: Brian Lam, Amy Nguyen ve Bryce Cronkite-Ratcliff

Ders Notları: JavaScript Syntax Referansı 

// FILE: randomCircles.js

const MIN_BALL_DIAMETER = 20;
const MAX_BALL_DIAMETER = 150;

// Adds a single random circle to the canvas
function addRandomCircle() {
    var ball = new GOval();
    var diameter = randomInt(MIN_BALL_DIAMETER, MAX_BALL_DIAMETER);
    ball.setColor(randomColor());
    ball.setFilled(true);
    ball.setSize(diameter, diameter);
    var randomX = randomInt(0, canvas.offsetWidth-diameter);
    var randomY =  randomInt(0, canvas.offsetHeight-diameter);
    ball.setLocation(randomX, randomY);
    canvas.add(ball);
}

// FILE: main.js
function run() {
    for (var i = 0; i < 10; i++) {
        addRandomCircle();
    }
}

Tesadüfi daireleri severim. O halde hadi birkaç tane yapalım!

Bu program bir web sitesi ile başlıyor. Kullanıcı her ekrana tıkladığında rastgele bir daire ekranda belirir.

Sizin göreviniz randomCircles.js içindeki addRandomCircle ve main.js içindeki run memtotlarını tamamlamak. İki dosyayı da "Notepad++" ile açın. Dosyaları "js" klasörünün içindeki "randomCircles" klasörünün içindeki "jsday1" klasöründe bulabilirsiniz. Bu fonksiyon, rastgele bir koordinatta rastgele boyur ve renkte bir daire yaratacaktır. Üretilen dairelerin her zaman ekranın tamamen içinde kaldığından emin olun!

Birinci Gün

Ekran açıldığında 20 adet rastgele daire ekrana gelsin.

İkinci Gün

Kullanıcı (siz) ekrana tıkladığında rastgele bir daire ekrana gelsin.

Not: Daireler çizebilmek için bu sayfanın altında belirtilen GOval referansındaki metotları kullanacaksınız. Referansa bir göz atmanızı öneririz!

Ek: Dairelerin kullanıcının tıkladığı noktada ortalanmasını sağlayın.

Debugging (Hata ayıklama)

Bir şeyler yanlışmış gibi gelirse Javascript Konsolunun bazen kullanışlı hata mesajları verdiğini unutmayın. Bu mesajlar hatanızı nasıl çözeceğinizi gösterir. Dolayısıyla onları kullanmayı alışkanlık haline getirmelisiniz!

GOval Referansı

Tesadüfi dairelerinizi yaratmak için aşağıdaki kullanışlı fonksiyonlara başvurabilirsiniz:

Yeni daire yaratma:

var myCoolOval = new GOval();

Dairenin özelliklerini belirleme:

// Dairenin rengini belirleme
myCoolOval.setColor(color);

// Dairenin yükseklik ve genişliğini belirleme
myCoolOval.setSize(width, height);

// Bu daireyi çevreleyen hayali dikdörtgenin sol üst köşesinin koordinatlarını belirleme
myCoolOval.setLocation(x,y);

Dairenizi ekrana getirme:

// ekran genişliği
canvas.offsetWidth; // returns 1000 (for example)

// ekran yüksekliği
canvas.offsetHeight; // returns 800 (for example)

// myCoolOval isimli dairenizi ekrana ekleme
canvas.add(myCoolOval);

Rastgelelilik

// min ve max değerleri (dahil) arasında rastgele bir sayı elde etmek
randomInt(min, max);

// rastgele bir renk elde etmek
randomColor();
